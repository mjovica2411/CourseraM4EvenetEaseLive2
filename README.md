# EventEase Blazor App

A simple Blazor WebAssembly application for event management.

## Features
- Event Card component with two-way binding
- Routing between multiple pages
- Registration form with validation
- Session state management
- Attendance tracker component

## Microsoft Copilot Summary
Microsoft Copilot assisted in the following development steps:

1. Generated the foundational Event Card component with fields and two-way data binding.
2. Helped implement and debug routing between Events, Register, and Dashboard pages.
3. Suggested validation improvements for the registration form.
4. Assisted with simple session state management using a scoped service.
5. Helped create the Attendance Tracker reusable component.
6. Provided debugging and optimization suggestions for cleaner reusable code.
